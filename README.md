# Food & Weight Tracker App

A minimalist and intuitive mobile application built with React Native for tracking daily nutrition and weight. The app allows users to easily log their meals, calculate calories based on portion sizes, and monitor their weight history.

## Key Features

* **Daily Calorie Goal:** Visual tracking of consumed calories against a daily limit.
* **Smart Food Search:** Built-in searchable database of products with automatic calorie calculation based on grams.
* **Weight Diary:** Log and track weight changes over time with date stamps.
* **Local Storage:** All user data (meals and weight history) is securely saved on the device using `AsyncStorage` for offline access.
* **Clean UI:** Modern, user-friendly interface with intuitive stack navigation.

## Tech Stack

* **Framework:** React Native
* **Navigation:** React Navigation (`@react-navigation/native-stack`)
* **Data Persistence:** `@react-native-async-storage/async-storage`
* **State Management:** React Context API & React Hooks
